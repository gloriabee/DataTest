# IBM HR Employees Attrition (EDL Process)

## Prequisties

- Python 3.12.\*
- PostgreSQL
- Kaggle API credentials
- Required Python packages
```
pip install pandas sqlalchemy python-dotenv kaggle
```
## Credential Data Format
```
DATABASE_URL=
KAGGLE_USERNAME=
KAGGLE_KEY=
```
## Usage

Run this script

```
python main.py
```
